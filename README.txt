Velnex WordPress Block Theme

This theme follows the default Twenty Twenty-Five style structure:

- style.css
- functions.php
- theme.json
- templates/front-page.html
- templates/index.html
- templates/page-services.html
- templates/page-opportunities.html
- parts/header.html
- parts/hero.html
- parts/firm.html
- parts/model.html
- parts/gap.html
- parts/people.html
- parts/services.html
- parts/opportunities.html
- parts/contact.html
- parts/footer.html
- assets/styles.css
- assets/script.js

Upload through WordPress:
Appearance > Themes > Add New > Upload Theme

Or upload by Hostinger File Manager:
Extract the folder into public_html/wp-content/themes/

Expected final path:
public_html/wp-content/themes/velnex-block-theme/templates/index.html

Edit sections:
Each major website section is separated in the parts folder.
